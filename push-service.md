# Как подключить пуш-уведомления

Для работы уведомлений нужно один раз настроить две вещи: сначала подготовить настройки в панели Remnawave, затем активировать уведомления в приложении.

---

## Шаг 1. Найдите или создайте секрет в панели Remnawave

Откройте файл `.env` вашей панели и найдите параметр `WEBHOOK_SECRET_HEADER`.

**Если он уже заполнен** — просто скопируйте его значение. Оно понадобится на следующем шаге.

> [!WARNING]
> Не меняйте существующий секрет. Если вы уже используете вебхуки, смена секрета сломает все, которые сейчас работают.

**Если он пустой** — сгенерируйте секрет и вставьте результат в `.env`:

```bash
openssl rand -hex 64
```

```env
WEBHOOK_SECRET_HEADER=сюда-вставьте-результат-команды-выше
```

---

## Шаг 2. Подключите приложение

Откройте приложение и перейдите в **Настройки → Соединение → Пуш-уведомления**.

Вставьте скопированный секрет в соответствующее поле и нажмите кнопку **Подключить**.

После подключения приложение выдаст вам **Webhook URL** — скопируйте его, он нужен для следующего шага. Ссылка будет выглядеть так:

```
https://your-domain.com/api/v1/webhooks/<panel-id>
```

---

## Шаг 3. Обновите настройки панели Remnawave

Вернитесь в файл `.env` и обновите параметры:

### `WEBHOOK_ENABLED`

Если значение `false` — смените на `true`:

```env
WEBHOOK_ENABLED=true
```

### `WEBHOOK_URL`

Если строка пустая — просто вставьте ссылку из приложения:

```env
WEBHOOK_URL=https://your-domain.com/api/v1/webhooks/<panel-id>
```

Если там уже есть другие ссылки — добавьте новую через запятую, не удаляя старые:

```env
WEBHOOK_URL=https://existing-webhook,https://another-webhook,https://your-domain.com/api/v1/webhooks/<panel-id>
```

---

## Шаг 4. Перезапустите панель и проверьте логи

Выполните команду — она остановит панель, запустит заново и откроет логи, чтобы убедиться что всё работает:

```bash
docker compose down && docker compose up -d && docker compose logs -f -t
```

Если всё в порядке, в логах не будет ошибок и панель успешно запустится. Для выхода из просмотра логов нажмите `Ctrl + C` — панель продолжит работать в фоне.

---

## 🔒 О безопасности

Все чувствительные данные — включая Webhook URL и все секреты — хранятся на вашем устройстве в зашифрованном виде. Никто не может получить к ним доступ без вашего ведома.
